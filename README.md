# GEMonitor
Intelligent environmental detection device based on Arduino and ESP8266.

# Feature
- Temperature and humidity detection
- Illumination Detection
- Human body detection
- Relay remote control
- Data on the cloud
- Network Interaction
- Flame Alarm
- OLED Display
